# 📝 Notes App

A responsive Notes App built using HTML, CSS, Bootstrap, and JavaScript. This application allows users to create, edit, delete, search, and save notes using the browser's Local Storage.

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Run the project using Live Server or open `index.html` in your browser.
4. Start creating and managing your notes.
